# Movie Explorer Application

## Project Overview

Movie Explorer is a web application that allows users to search, discover, and explore movies using real-time data from an external movie database API. The application helps users find movies based on keywords, genres, and popularity while also allowing them to save favorite movies for later viewing.

This project was developed as part of the WDD 330 Web Frontend Development course. The goal of the project is to demonstrate the use of modern JavaScript frameworks, API integration, responsive design, and user interaction features.

---

## Features

- Search for movies by title
- View detailed movie information
- Filter movies by category or genre
- Add and remove movies from favorites
- Responsive layout for mobile and desktop
- Dynamic content rendering from API data
- Navigation between application views
- Error handling for failed API requests

---

## Technologies Used

This application was built using:

- React
- TypeScript
- Vite
- Tailwind CSS
- External Movie API (TMDB)
